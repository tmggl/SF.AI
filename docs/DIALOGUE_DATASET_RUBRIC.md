# DIALOGUE_DATASET_RUBRIC.md

## SF.AI — معيار جودة بيانات الحوار

هذا المعيار يساعدنا نقرر هل العينة تصلح لتدريب نموذج SF.AI السيادي.

---

## العينة الذهبية `gold`

العينة `gold` يجب أن تكون:

- طبيعية في الفصحى أو السعودية.
- قصيرة أو متوسطة، بلا حشو.
- تحتوي user و assistant.
- رد assistant مفيد ومتواضع، لا يدّعي قدرات غير موجودة.
- بلا معلومات شخصية.
- بلا نصائح حساسة.
- مصدرها واضح.

مثال:

```json
{
  "domain": "chat",
  "lang": "ar",
  "messages": [
    {"role": "user", "content": "وش تقدر تسوي؟"},
    {"role": "assistant", "content": "أقدر أفهم الفصحى واللهجة السعودية وأوجّه طلبك داخل النظام. التوليد الذكي الكامل يحتاج تدريب النموذج السيادي."}
  ],
  "provenance": {
    "source": "sami-authored",
    "license": "user-provided",
    "language": "ar",
    "dialect": "saudi",
    "quality": "gold"
  }
}
```

---

## العينة الفضية `silver`

صالحة غالبًا، لكن فيها شيء يحتاج تحسينًا لاحقًا:

- طويلة قليلًا.
- الأسلوب أقل طبيعية.
- اللهجة مفهومة لكن ليست مثالية.
- الرد صحيح لكنه يحتاج اختصارًا.

---

## العينة البرونزية `bronze`

صالحة بنيويًا فقط:

- قصيرة جدًا.
- مكررة.
- مفيدة لتغطية كلمة أو intent، لكنها ليست مثالًا ممتازًا للحوار.

لا نبني جودة النموذج عليها وحدها.

---

## العينة المرفوضة

ارفض العينة إذا:

- لا يوجد `provenance`.
- اللهجة ليست `msa` أو `saudi`.
- لا يوجد assistant response.
- فيها كلام حساس بلا سياسة.
- فيها LLM-generated text.
- فيها نسخ من مصدر لا نملك حق استخدامه.
- فيها وعد كاذب: مثل "أقدر أشخّص مرضك" أو "أقدر أكتب كود كامل الآن" قبل تفعيل المجال.
