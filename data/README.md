# data/

البيانات الخام والمنظمة المستخدمة لبناء SF.AI.

## الهيكل

```
data/
├── corpus/             # كل بيانات SF.AI الخام والمنظمة
│   └── chat/           # corpus الحوار (يُعبَّأ في Phase 5)
│       ├── raw/        # ملفات خام كما وردت من المصدر
│       ├── cleaned/    # بعد التنظيف
│       └── jsonl/      # JSONL جاهز للاستخدام
└── indexes/            # فهارس بحث / lookup tables
```

## قواعد البيانات

1. **لا تضع** بيانات مولدة من LLM خارجي داخل corpus السيادي.
2. **لا تضع** بيانات بدون توثيق المصدر والترخيص.
3. **لا تضع** أي ملف يحوي معلومات حساسة شخصية.
4. **كل** ملف يجب أن يُرفق بـ `provenance.json` (لاحقًا) يصف:
   - المصدر.
   - تاريخ الجلب.
   - الترخيص.
   - اللغة / اللهجة.
   - عدد العينات.
5. **UTF-8** فقط.
6. **JSONL** هي الصيغة المعيارية بعد التنظيف.

## ملاحظات

- Phase 1: المجلد فارغ. لا تبدأ في إدخال بيانات الآن.
- Phase 5 يضيف schemas/validators/loaders.
- المستخدم هو من يقرر متى تُدخَل بيانات الحوار.
